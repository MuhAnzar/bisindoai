Templat Pengujian (Test Scripts)

Dokumen ini berisi tabel pengujian lengkap yang siap digunakan untuk verifikasi manual. Kolom Hasil dan Catatan dikosongkan agar Anda dapat mengisinya saat melakukan pengujian.

1. Pengujian Black Box (Fungsionalitas Sistem)

ID Fitur Skenario Uji Langkah Pengujian Hasil yang Diharapkan Status (Pass/Fail) Catatan / Temuan
A. MANAJEMEN AKUN
A1 Login Login dengan kredensial valid 1. Buka halaman login
2. Masukkan email & password benar
3. Klik Login		 Masuk ke dashboard user
A2 Login Login dengan password salah 1. Buka halaman login
2. Masukkan email benar & password salah
3. Klik Login		 Muncul pesan error "Password salah"
A3 Logout Logout dari sistem 1. Klik ikon profil
2. Klik tombol Logout		 Kembali ke halaman login
B. NAVIGASI
B1 Sidebar Akses Menu "Latihan" Klik menu "Latihan" di sidebar Halaman Latihan terbuka
B2 Sidebar Akses Menu "Riwayat" Klik menu "Riwayat" di sidebar Halaman Riwayat terbuka
C. MODE LATIHAN (CORE)
C1 Setup Kamera Izin Kamera Buka halaman latihan Browser meminta izin akses kamera
C2 Setup Kamera Tampilan Video Izinkan akses kamera Video feed tampil di layar
C3 Mode Abjad Switch Mode Klik tab "Abjad" UI input teks muncul (A-Z)
C4 Mode Abjad Input Validasi Ketik angka "123" Input menolak angka (validasi A-Z only)
C5 Mode Kata Switch Mode Klik tab "Kata" Grid kartu kata muncul
C6 Mode Kata Multi-select Klik 3 kartu kata berbeda Tombol mulai menunjukkan "(3) item"
C7 Mode Kalimat Switch Mode Klik tab "Kalimat" UI penyusun kalimat (word builder) muncul
C8 Mode Kalimat Tambah Kata Klik kartu kata di picker Kata masuk ke baris kalimat
C9 Mode Kalimat Tambah Ejaan Ketik manual dan enter Ejaan per huruf masuk ke baris kalimat
D. PROSES DETEKSI (AI)
D1 Visualisasi Skeleton Hand (Kata) Gerakkan tangan cepat di Mode Kata Garis skeleton mengikuti tangan tanpa lag
D2 Feedback Deteksi Benar Isyaratkan kata target dengan benar Visual feedback hijau & lanjut ke kata berikutnya
D3 Feedback Deteksi Salah Isyaratkan kata sembarang Visual feedback salah/tidak ada respon
D4 Komplesi Selesai Latihan Selesaikan semua target kata Modal "Latihan Selesai" & Skor muncul
E. RIWAYAT
E1 Pencatatan Simpan Data Selesaikan latihan Data sesi baru muncul di database/tabel
E2 Tampilan Badge Akurasi Buka menu Riwayat Sesi terakhir tampil dengan badge (Excellent/Good)
F. VALIDASI INPUT
F1 Input Abjad Huruf Kecil Masukkan huruf "a, b, c" (kecil) pada mode Abjad Sistem otomatis mengonversi menjadi kapital "A, B, C"
F2 Input Abjad Angka & Simbol Ketik "123!@#" pada input teks Input ditolak atau tidak muncul di layar (Input sanitization)
F3 Input Abjad Input Kosong Kosongkan kolom input lalu klik "Mulai" Muncul pesan error/toast "Masukkan kata target!"
F4 Input Abjad Spasi Ketik "A B" (dengan spasi) Spasi dihapus atau diabaikan (hanya huruf)
F5 Input Kata Kata Tidak Valid Ketik kata manual yang tidak ada di database (Cth: "Xyzz") Tombol mulai disable & Error "Kata tidak tersedia"
F6 Input Kalimat Format Campuran Masukkan kombinasi Kata + Ejaan Sistem menerima keduanya dalam satu baris kalimat
